Entendiendo el sistema de columnas

Bootrstrap tiene un sistema de 12 grillas responsive, esto quiere decir que son escalables y adaptables a la pantalla del dispositivo que utilizamos para visualizar el sitio web que construimos. Este párrafo está en una col-sm-12, eso quiere decir que a una resolución mayor a 768px nuestro contenido abarcará el 100% del elemento que lo contiene.

Este párrafo tiene como atributo la clase visible-xs. Al igual que el contenedor que acaba de aparecer.
